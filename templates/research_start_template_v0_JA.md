## ② フェーズ別ドリル（研究スタートキャンバス）

### Phase 01 ─ ORCID登録：研究者ライセンスの発行

*ORCID は研究者の国際的な固有識別番号 ──「研究者の LinkedIn」。これがないと存在しないに等しい。*

- [ ] orcid.org で登録し ORCID iD を取得した（URL: 〔 https://orcid.org/0000-　 〕）
- [ ] 所属機関・研究分野・学歴・業績を入力してプロフィールを完成させた
- [ ] Google Scholar / ResearchGate / Semantic Scholar を整備し ORCID と連携した

完了日： 〔　〕

### Phase 02 ─ 問い・仮説の言語化：A4で1〜2枚

*これが論文の Introduction の共通情報源になります。いつでも更新してOK。*

- **W  What?** どんな現象・問題に興味があるか（「〜が面白い」「〜が不思議」を素直に）
  〔　〕
- **Y  Why?** なぜ重要か・誰が困っているか（社会・自然・世界のどのテーマか明確に）
  〔　〕
- **H  How?** どう解決できると思うか＝仮説（DSアプローチで「〜ではないか」）
  〔　〕

### Phase 03 ─ LP・名刺・ブランディング：磁石になろう

*「XをしているYさん」ブランドで、情報・紹介・共同研究・ユーザーを引き寄せる磁石になる。*

- [ ] LP を作成した（URL: 〔　〕）── Coming Soon で十分
- [ ] 事前登録フォームを設置した（興味ユーザーの定量データが研究根拠になる）
- [ ] 名刺を作成した（氏名・所属・研究テーマ・LP URL・QRコード）
- [ ] Webアプリ（MVP）を公開し LP からリンクした（URL: 〔　〕）
